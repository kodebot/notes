# Common matrix transformations

## Identity matrix

Identity matrix is a matrix that does nothing

$$
\begin{bmatrix}1 & 0 \\
0 & 1\end{bmatrix}
$$

It has ones on leading diagonal and zeros in other places.


## Scaling matrix

Identity matrix is a matrix that does nothing

$$
\begin{bmatrix}1 & 0 \\
0 & 1\end{bmatrix}
$$

It has ones on leading diagonal and zeros in other places.
